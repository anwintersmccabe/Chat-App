# Chat-App

A python implementation of a client-server chat application 
## Features

- Can host up to 5 users
## Prerequisites 

Python3 is required
```zsh
$ python --version
```
## Usage

Navigate to the chat-app directory and run the server
```zsh
$ python server.py
```
Example HOST and PORT
```zsh
HOST: 127.0.0.1
PORT: 33000
```
Run the client to connect a user
```zsh
$ python client.py
```
